Паттерн проектирования "Фабрика" (Factory Method) используется для создания объектов,
не указывая конкретный класс создаваемого объекта. Он позволяет создавать объекты через
интерфейс, оставляя реализацию конкретных классов на стороне подклассов.

Основные компоненты паттерна "Фабрика":

Продукт: Интерфейс или абстрактный класс, который определяет тип объектов, создаваемых фабрикой.
Конкретные продукты: Классы, реализующие интерфейс продукта.
Фабрика: Интерфейс или абстрактный класс, который объявляет метод для создания продукта.
Конкретные фабрики: Классы, которые реализуют метод создания продукта и возвращают экземпляры конкретных продуктов.
